手机和桌面混合情况

Bootstrap 3 栅格系统有四套class: xs (phones), sm (tablets), md (desktops), and lg (larger desktops). 你可以使用他们中任意的组合来创建更加动态灵活的布局

每一套 class 都能够很好的扩展，这意味着如果你想为 xs 和 sm 设置同样的宽度，只需要指定 xs

.col-xs-12 .col-md-8
.col-xs-6 .col-md-4
.col-xs-6 .col-md-4
.col-xs-6 .col-md-4
.col-xs-6 .col-md-4
.col-xs-6
.col-xs-6

在特定的临界点重置offsets, pushes,pulls

.col-sm-5 .col-sm-offset-2 .col-md-6 .col-md-offset-0
